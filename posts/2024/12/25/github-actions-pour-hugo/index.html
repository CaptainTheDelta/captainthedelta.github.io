    GitHub pages#

    Création d’une page github.io#

    Comme vous le savez, ou pas, GitHub permet d’héberger un site statique par utilisateur. Pour en bénéficier, il faut créer le répertoire [username].github.io. Par exemple, dans mon cas, on passe de CaptainTheDelta à captainthedelta.github.io.

    Préparation du répertoire#

    Pour séparer le répertoire d’hugo des pages générées, on crée la branche gh-pages.

    [!warning] Je n’invente pas le nom, il est réutilisé par le script que nous utiliserons plus tard. Je vous invite à utiliser le même nom.
